Tabiq

Privacy Policy

Effective date: June 2025

Tabiq is a browser extension for Chrome and Microsoft Edge that helps you organize browser tabs by type and AI-powered topic grouping. This policy explains what data Tabiq accesses, how it is used, and how it is stored.

1. Data we access

Tabiq accesses the following information about your open browser tabs:

  • Tab titles
  • Tab URLs
  • Tab active/inactive status
  • Window ID (to allow switching focus between windows)

This data is used solely to display and organize your tabs within the extension. It is never sold, shared with third parties, or used for advertising.

2. Local storage

Tabiq stores the following data locally on your device using Chrome's built-in storage:

  • Your Anthropic API key (if provided) — stored locally and never transmitted except directly to the Anthropic API
  • AI-generated topic groupings — cached to avoid repeated API calls
  • Custom tab ordering preferences set via drag and drop

This data remains on your device and is never transmitted to any server operated by Tabiq.

3. AI-powered topic grouping

If you choose to use the "By topic" feature and provide a Claude API key, Tabiq will send your tab titles and domain names (not full URLs) to the Anthropic API to generate topic groupings. This is an optional feature.

Data sent to Anthropic is governed by Anthropic's Privacy Policy. Tabiq has no control over how Anthropic processes this data.

Your API key is stored only on your device. Tabiq never stores your API key on any external server.

4. Data we do not collect

Tabiq does not:

  • Collect or transmit your browsing history
  • Read the content of web pages you visit
  • Use cookies or tracking technologies
  • Collect personally identifiable information
  • Share any data with advertisers or data brokers
  • Operate any backend server or database

5. Permissions

Tabiq requests the following browser permissions:

  • tabs — to read the titles, URLs, and status of your open tabs
  • storage — to save your preferences and API key locally on your device

No other permissions are requested. Tabiq does not request access to your browsing history, bookmarks, downloads, or any other browser data.

6. Children's privacy

Tabiq is not directed at children under the age of 13 and does not knowingly collect any information from children.

7. Changes to this policy

If this privacy policy changes materially, we will update the effective date at the top of this document and note the changes in the extension's release notes.

8. Contact

If you have any questions about this privacy policy, please reach out via the support link on the Tabiq Chrome Web Store or Microsoft Edge Add-ons listing.
